# Simple Java Blockchain ⛓️

Простая реализация блокчейна на Java, демонстрирующая ключевые принципы работы криптовалют.

## Особенности
* **Proof of Work:** Алгоритм майнинга с настраиваемой сложностью.
* **Цифровые подписи:** Безопасность транзакций с помощью ECDSA.
* **Система вознаграждений:** Начисление монет майнеру за каждый найденный блок.
* **Расчет баланса:** Динамическое сканирование цепочки для определения остатка средств.

## Как запустить
1. Склонируйте репозиторий.
2. Откройте проект в любой IDE (IntelliJ IDEA, Eclipse).
3. Запустите файл `BlockchainProject.java`.

## Структура проекта
- `Block.java` — Логика хранения данных и связи блоков.
- `Transaction.java` — Логика переводов и проверки подписей.
- `Wallet.java` — Генерация ключей и адресов.
- `StringUtil.java` — Инструмент для SHA-256 шифрования.
